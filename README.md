# AI-BASED-CROWD-MANAGEMENT-SYSTEM
This project uses computer vision and machine learning techniques to detect and count people from video frames. It includes advanced statistical visualizations such as box plots, scatter plots, and correlation matrices to help understand crowd patterns over time.

### 🚀 Features

- ✔️ Real-time crowd detection from CCTV/video
- ✔️ Crowd counting using AI/Deep Learning
- ✔️ Visualization dashboard (Box Plot, Scatter Plot, Correlation Matrix)
- ✔️ Trend analysis using lag values and moving averages
- ✔️ Alerts for high-density crowd situations
- ✔️ Clean modular code structure

### 🔧 Tech Stack

- Python (OpenCV, NumPy, Pandas)
- Deep Learning (YOLO / MobileNet / OpenCV DNN)
- Matplotlib & Seaborn (Visualizations)
- Jupyter Notebook (Exploration)

### 📈 Sample Output Visualizations

- Box Plot — Crowd Distribution
- Scatter Plot — Time vs Crowd Size
- Correlation Matrix — Trend Prediction
##### DATA SET LINK- https://www.kaggle.com/datasets/ubaydulloasatullaev/crowd-detection-video?select=pexels_videos_1677252+%282160p%29.mp4
